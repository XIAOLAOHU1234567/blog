回路矩阵

回路计算

  1. 完全回路矩阵($C_e$)：形如下面例子：这个矩阵可以表示连通图中所有的回路。不过，矩阵中包含冗余的信息，并且不易计算。根据题意，图中最多有$(2^{m-n+1}-1)$棵树。
  2. 基本回路矩阵：首先找到图的一棵支撑树，然后对于每一条余树边，找出其对应的回路并且写出。可以得到，这样的矩阵的秩为[m-(n-1)=m-n+1].这个矩阵记作$C_f$
    • 对这个定理的证明：使用Sylvester引理。
    • 如果关联矩阵和完全回路矩阵的边次序是相同的，那么，[BC_e^T=0]
  3. 回路矩阵：图的任意$(m-n+1)$条独立回路构成的矩阵，由上面的总结可知：$rank = m-n+1$.记作$C$. 由上面的论述可知,回路矩阵$C$具有以下性质:
    1. [r(C) = m-n+1]
    2. [BC^T=0]前提要求是B和C边的顺序一致. 注意注意:这里B是关联矩阵你,而不是基本关联矩阵!(==这里基本关联矩阵是否可以使用==?) 在这里只有最后最重要的定理使用的是基本关联矩阵.
    3. 由于基本回路矩阵也是回路矩阵,那么,如果边的顺序一致,他们两个是相抵的.也就是说, 存在非奇异变换矩阵$P$, s.t. [C=PC_f]

  4. 联系: 如何由基本关联矩阵求解基本回路矩阵?

    1. 定理1: 记一张图的基本关联矩阵为$B$, 基本回路矩阵为$C$, 通过分块变换使得两个矩阵拆分为: [C= [I \space C{1f} ], B = [B{11} \space B_{12}]]
      • 这里B12为支撑树的边,11为余树边,顺序与构成回路的顺序一致.
      • 那么有: [C{1f} = -B{11}B_{12}^{-T}]
    2. 定理2: 连通图的回路矩阵的m-n+1阶子阵可逆 $\Leftrightarrow$ 这些列构成一棵余树, 也就是去掉的列中没有基本回路.

    3. 割集计算

  5. 如果有一组边的集合$S$满足:

    1. $(V, E-S)$的连通支恰好比$(V, E)$的连通支多1.
    2. 对于任何的$S’ \subset S$, 这个集合不满足1条件.那么称S为割集, 记作S
  6. 割集的表示:割集矩阵
    1. 割集可以规定一个方向
    2. 割集矩阵, 形如
      1. 完全割集矩阵
  7. 基本割集矩阵: 考虑树枝
    1. 考虑连通图G和支撑树T
    2. 对于T的树干$e_i$, 总是存在一个割集S, 包含$e_i$和其他余树边, 并且这个割集S和这条树干方向相同.
    3. 此时$S_i$被称为基本割集
    4. 由全部基本割集构成的矩阵称为基本割集矩阵, 记作$S_f$.[r(S_f) = n - 1]
  8. 割集矩阵和回路矩阵的关系;
    1. 当连通图的完全割集矩阵$S_e$和完全回路矩阵$C_e$的边顺序一致的时候, 由[S_eC_e^T=0]
    2. [r(S_e)=n-1]
  9. 割集矩阵和其性质,3条,同上回路矩阵
  10. 定理:
    1. [Sf=[S{f11} \space I], Cf = [I \space C{f12}]] [S{f11} = -C{f12}^T]
    2. [S{f11} = B{12}^{-1}B_{11}]
    3. 注意拆分方法: 回路先I

Huffman-Tree

由huffman算法得到的二叉树称为最优二叉树, WPL最小.复杂度$O(n\log n)$

最短树: Kruskal算法 & Prim算法

  1. Kruskal:求解思路
    1. 首先向T中加入目前的最短边
    2. 如果加入$e_i$之后出现回路,那么从图中删去
    3. until $|T|=n-1$
    4. $O(m+plogm)$
  2. 如何判定最短树? $\Leftrightarrow$
    • 任意的余树边都比树枝边长
  3. Prim算法;
    1. 每次都加入两个点集中的最短边
    2. $O(n^2)$

经验总结

  1. 正规变换的三条性质：
    1. 对应正规矩阵
    2. $T(v) = T^*(v)$
    3. T和它的伴随变换具有相同的特征向量，并且这两个特征向量所对应的特征值互为共轭复数。
    4. 属于不同特征值的特征向量互相正交。几何证明：$(Ax,y)=(x,A^*y)$
  2. 循环矩阵的特征值和特征向量的计算：参见线性算子.md
  3. 酉矩阵的特征值的模长都是1.
  4. 对于正规矩阵A，如果V是A的不变子空间，那么$V^\perp$是$A^H$的不变子空间.但是由于A是正规阵, 那么V也是$A^H$的不变子空间. 这里是使用分块对角阵证明的结论, 使用了取trace的技巧. ==非常重要==
  5. 别忘了正规阵可以酉对角化并且可以谱分解, 这一点很重要.
  6. 是否掌握了Hamilton-Cayley定理的证明?
  7. 一个复矩阵正规 $\Leftrightarrow$ 可以分解为 $A=B+iC$ , 其中B, C都自伴, 并且可交换. (构造法证明)
  8. 如果T正规, S和T可交换, 那么S和 $T^*$ 也可交换(使用谱分解证明).

正交投影变换

  1. 如果实方阵满足：
    1. $A^T = A$
    2. $A^2=A$
      那么，A被称为正交投影阵。
  2. 特别地，应该如何计算A对其列空间的投影？
    1. 如果A是列满秩矩阵，那么$A^TA$就是可逆的，因此投影矩阵$P_{C(A)}=A(A^TA)A^T$。
    2. 如果A本身就是正交投影矩阵。那么，A就是对其列空间进行投影的矩阵。
      如何理解上述说法？
      对于空间中的向量，只有三种方向。在投影平面上的、与投影平面正交的、普通向量。$\vec{x}是空间中任意一个向量，而$A \vec{x}$是列空间中的向量.令A去左乘$(\vec{x} - A \vec{x})$,可以知道结果为零.因此知道了投影就是向列空间的平面进行的.证明方法可以做一个直和正交分解.
  3. 性质:给定一个n阶方阵A,如果满足$A^2=A$, 那么A可以被相似对角化,使得$\Lambda = diag(1,1,…,0,0,…)$. 这个虽然不是正交投影, 但是仍然是投影.也可以用”三类向量”来理解. 因此, A必须有且只能.
  4. 补充:如何理解相似对角化?
    • 将矩阵视为一种集合变换,特征向量的方向便是变换之后方向不会改变的方向. $P^{-1}$相当于将矩阵翻译为特征坐标系,再乘$P$将其又翻译回来. 因此,投影代表在平面上的方向都不变,正交于平面方向的向量都归零.
    • 如果要证明的话可以先把P弄到左边来再做打算
  5. 矩阵的证明中取特征值是常用的手法.
  6. 在非标准正交基下的投影矩阵未必对称.
  7. 内蕴的正交投影定义:
    1. 设$V=V_1 \oplus V_2 \oplus V_3 \oplus ….$, 如果有$(\forall x)(x \in R^n \to E_i(x) \in V_i), i = 1, 2, ….$. 如果还有$E_i^2 = E_i, E_i E_j=0, \Sigma E_i = I_n$. 那么E就是正交投影变换.
    2. 补充: 线性变换加法的定义:$(\sigma_1 + \sigma_2)(x)=\sigma_1(x)+\sigma_2(x)$

线性算子

  1. 伴随算子(共轭算子).
    1. 定义:设V为内积空间,T:V$\to$V为线性变换,若$\forall \alpha, \beta, (T(\alpha), \beta) = (\alpha, T^*(\beta))$, 那么这两个变换互为伴随变换. 矩阵上的表现为转置或者共轭转置.
    2. 伴随变换的性质:
      1. $(T_1+T_2)^=T_1^ + T_2^*$
      2. $(T_1 \circ T_2)^=T_2^ \circ T_1^*$
      3. 不变子空间互相正交,特征值互相共轭.
    3. 看到标准正交基之后,在变换过程中可以考虑内积.
  2. 保积变换:容易理解.一定是正交矩阵.性质如下:
    1. 将标准正交基映射为另一组标准正交基
    2. 符合为id
    3. 不改变模长和夹角
    4. 酉矩阵
  3. 正规算子
    1. 如果两个线性变换满足:$T^T=TT^$, 那么这个变换是正规变换
    2. 性质:
      1. 正规矩阵
      2. 变换与其伴随模长相等
      3. 相同特征向量,特征值共轭,当且仅当
      4. 必正交
  4. 自伴算子

相关定理

  1. Shcur定理:在n维酉空间内, 对于线性变换F, 必然存在一组标准正交基, 使得这个线性变换的表示矩阵为上三角阵.(归纳法)
    ==注意:Schur定理等价描述: 任何复矩阵酉相似于一个上三角阵.使用换基的过渡矩阵证明的==
  2. 正规变换酉对角化定理:正规当且仅当存在标准正交基使得T在这组基下的矩阵为酉对角阵.
  3. 谱分解:设V是有限维的内积空间, 则:$E=\lambda_1E_1+\lambda_2E_2+…$
  4. 实正规变换: 这个定理告诉我们,无论多么复杂的正规变换,我们都可以拆分成沿着向量的伸长和沿着某几个平面的旋转,并且他们绝对正交.
  5. 换基相当于改变视角, 别忘了换基过度矩阵的公式.
  6. 要深刻理解正规变换的酉对角化定理和real正规变换定理的关系。
