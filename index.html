高代作业总结3

经验总结

  1. 正规变换的三条性质：
    1. 对应正规矩阵
    2. T(v)=T∗(v)T(v) = T^*(v)T(v)=T∗(v)
    3. T和它的伴随变换具有相同的特征向量，并且这两个特征向量所对应的特征值互为共轭复数。
    4. 属于不同特征值的特征向量互相正交。几何证明：(Ax,y)=(x,A∗y)(Ax,y)=(x,A^*y)(Ax,y)=(x,A∗y)
  2. 循环矩阵的特征值和特征向量的计算：参见线性算子.md
  3. 酉矩阵的特征值的模长都是1.
  4. 对于正规矩阵A，如果V是A的不变子空间，那么V⊥V^\perpV⊥是AHA^HAH的不变子空间.但是由于A是正规阵, 那么V也是AHA^HAH的不变子空间. 这里是使用分块对角阵证明的结论, 使用了取trace的技巧. 非常重要
  5. 别忘了正规阵可以酉对角化并且可以谱分解, 这一点很重要.
  6. 是否掌握了Hamilton-Cayley定理的证明?
  7. 一个复矩阵正规⇔\Leftrightarrow⇔可以分解为A=B+iCA=B+iCA=B+iC, 其中B, C都自伴, 并且可交换. (构造法证明)
  8. 如果T正规, S和T可交换, 那么S和T∗T^*T∗也可交换(使用谱分解证明).

正交投影变换

  1. 如果实方阵满足：
    1. AT=AA^T = AAT=A
    2. A2=AA^2=AA2=A
      那么，A被称为正交投影阵。
  2. 特别地，应该如何计算A对其列空间的投影？
    1. 如果A是列满秩矩阵，那么ATAA^TAATA就是可逆的，因此投影矩阵PC(A)=A(ATA)ATP_{C(A)}=A(A^TA)A^TPC(A)​=A(ATA)AT。
    2. 如果A本身就是正交投影矩阵。那么，A就是对其列空间进行投影的矩阵。
      如何理解上述说法？
      对于空间中的向量，只有三种方向。在投影平面上的、与投影平面正交的、普通向量。x⃗是空间中任意一个向量，而\vec{x}是空间中任意一个向量，而x是空间中任意一个向量，而A \vec{x}是列空间中的向量.令A去左乘是列空间中的向量.令A去左乘是列空间中的向量.令A去左乘(\vec{x} - A \vec{x})$,可以知道结果为零.因此知道了投影就是向列空间的平面进行的.证明方法可以做一个直和正交分解.
  3. 性质:给定一个n阶方阵A,如果满足A2=AA^2=AA2=A, 那么A可以被相似对角化,使得Λ=diag(1,1,...,0,0,...)\Lambda = diag(1,1,...,0,0,...)Λ=diag(1,1,...,0,0,...). 这个虽然不是正交投影, 但是仍然是投影.也可以用"三类向量"来理解. 因此, A必须有且只能.
  4. 补充:如何理解相似对角化?
    • 将矩阵视为一种集合变换,特征向量的方向便是变换之后方向不会改变的方向. P−1P^{-1}P−1相当于将矩阵翻译为特征坐标系,再乘PPP将其又翻译回来. 因此,投影代表在平面上的方向都不变,正交于平面方向的向量都归零.
    • 如果要证明的话可以先把P弄到左边来再做打算
  5. 矩阵的证明中取特征值是常用的手法.
  6. 在非标准正交基下的投影矩阵未必对称.
  7. 内蕴的正交投影定义:
    1. 设V=V1⊕V2⊕V3⊕....V=V_1 \oplus V_2 \oplus V_3 \oplus ....V=V1​⊕V2​⊕V3​⊕...., 如果有(∀x)(x∈Rn→Ei(x)∈Vi),i=1,2,...(\forall x)(x \in R^n \to E_i(x) \in V_i), i = 1, 2, ...(∀x)(x∈Rn→Ei​(x)∈Vi​),i=1,2,.... 如果还有Ei2=Ei,EiEj=0,ΣEi=InE_i^2 = E_i, E_iE_j=0, \Sigma E_i = I_nEi2​=Ei​,Ei​Ej​=0,ΣEi​=In​. 那么E就是正交投影变换.

线性算子

  1. 伴随算子(共轭算子).
    1. 定义:设V为内积空间,T:V→\to→V为线性变换,若∀α,β,(T(α),β)=(α,T∗(β))\forall \alpha, \beta, (T(\alpha), \beta) = (\alpha, T^*(\beta))∀α,β,(T(α),β)=(α,T∗(β)), 那么这两个变换互为伴随变换. 矩阵上的表现为转置或者共轭转置.
    2. 伴随变换的性质:
      1. (T1+T2)∗=T1∗+T2∗(T_1+T_2)^*=T_1^* + T_2^*(T1​+T2​)∗=T1∗​+T2∗​
      2. $(T_1 \circ T_2)*=T_2* \circ T_1^*
      3. 不变子空间互相正交,特征值互相共轭.
    3. 看到标准正交基之后,在变换过程中可以考虑内积.
  2. 保积变换:容易理解.一定是正交矩阵.性质如下:
    1. 将标准正交基映射为另一组标准正交基
    2. 符合为id
    3. 不改变模长和夹角
    4. 酉矩阵
  3. 正规算子
    1. 如果两个线性变换满足:T∗T=TT∗T^*T=TT^*T∗T=TT∗, 那么这个变换是正规变换
    2. 性质:
      1. 正规矩阵
      2. 变换与其伴随模长相等
      3. 相同特征向量,特征值共轭,当且仅当
      4. 必正交
  4. 自伴算子

相关定理

  1. Shcur定理:在n维酉空间内, 对于线性变换F, 必然存在一组标准正交基, 使得这个线性变换的表示矩阵为上三角阵.(归纳法)
    注意:Schur定理等价描述: 任何复矩阵酉相似于一个上三角阵.使用换基的过渡矩阵证明的
  2. 正规变换酉对角化定理:正规当且仅当存在标准正交基使得T在这组基下的矩阵为酉对角阵.
  3. 谱分解:设V是有限维的内积空间, 则:E=λ1E1+λ2E2+...E=\lambda_1E_1+\lambda_2E_2+...E=λ1​E1​+λ2​E2​+...
  4. 实正规变换: 这个定理告诉我们,无论多么复杂的正规变换,我们都可以拆分成沿着向量的伸长和沿着某几个平面的旋转,并且他们绝对正交.
  5. 换基相当于改变视角, 别忘了换基过度矩阵的公式.
  6. 要深刻理解正规变换的酉对角化定理和real正规变换定理的关系。
